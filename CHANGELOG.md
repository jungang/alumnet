# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-05

### Added

- Initial open source release of AlumNet
- AI-powered alumni exhibition kiosk (Vue 3 + Three.js)
- Admin dashboard (Vue 3 + Element Plus)
- RESTful API backend (Express + TypeScript)
- RAG-powered natural language search with Qdrant
- PostgreSQL + pgvector for data and embeddings
- Docker Compose for one-command deployment
- Comprehensive test suite with Vitest
- CI/CD with GitHub Actions

### Features

- Alumni Galaxy: 3D interactive star map visualization
- Time Corridor: chronological photo gallery
- AI Smart Search: natural language alumni queries
- Message Board: interactive alumni greetings wall
- System Dashboard: usage analytics and backup management
